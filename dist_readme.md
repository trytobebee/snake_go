# 🎮 贪吃蛇游戏：快速开始指南 (Quick Start Guide)

欢迎体验 Snake Go v3.0！本指南旨在帮助非技术用户快速启动游戏。

## 一切的开始unzip下载到的zip包

---

## 🚀 方式 A：网页版 (推荐)
**网页版支持更精美的画面、声音效果以及手机操作。**

### 1. 启动服务器 (根据系统选择)
在 `dist` 文件夹中找到对应你系统的程序并运行：
- **Windows**: 双击 `webserver_windows.exe`
- **Mac (M1/M2/M3)**: 运行 `webserver_mac_arm64`
- **Mac (Intel)**: 运行 `webserver_mac_amd64`
- **Linux**: 运行 `webserver_linux`

### 2. 进入游戏
- **本电脑玩**: 打开浏览器，输入 👉 `http://localhost:8080`
- **📱 手机或其他电脑玩**: 
  1. 确保手机与电脑连接在**同一个 Wi-Fi**。
  2. 找到你电脑的 **局域网 IP 地址**：
     - **Windows**: 在终端输入 `ipconfig`，找 "IPv4 地址"（通常是 `192.168.x.x`）。
     - **Mac**: 按住 `Option` 键点击顶部 Wi-Fi 图标，即可看到 IP 地址。
  3. 在手机浏览器输入：`http://你的IP地址:8080` (例如 `http://192.168.1.5:8080`)

## 🎮 操作说明

### 💻 键盘操作：
- **方向键** 或 **W/A/S/D**: 控制转向。
- **长按方向键**: 触发 **🚀 加速 (Boost)**。
- **空格键 / P**: 暂停或继续。
- **R**: 死亡后重新开始。
- **A**: 开启/关闭 **🤖 AI 自动驾驶** (仅限网页版)。

### 📱 手机操作 (连接同一WiFi后通过IP访问):
- **虚拟按键**: 点击转向。
- **长按按键**: 触发加速。
- **机器人图标**: 切换自动驾驶。

---

## 💡 常见问题
- **没声音？** 网页版需在进入后点击一次页面方可激活音效。
- **打不开？** Mac 用户如果提示“无法验证开发者”，请在系统设置 -> 隐私与安全性中点击“仍要打开”。
